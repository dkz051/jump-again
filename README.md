# Jump Again

2020 春《数字逻辑设计》课程项目：体感平台跳跃游戏

## 项目灵感

软院大一上程设大作业：平台跳跃游戏（例如 Celeste, I wanna 系列）

## 操作方法

体感模拟键盘。

* 左/右：左/右横滚旋转
* 跳：快速向上移动
* 二段跳：再次快速向上移动
